SOOTER GAME - README (Compact)

Game Type: 3‑lane console dodging game in C
Goal: Move left/right to avoid falling obstacles while increasing score.

Controls

-   Left Arrow → Move left
-   Right Arrow → Move right

Features

-   Random 1–2 lane obstacles
-   Live score tracking
-   High‑score system (saved in highscore.txt)
-   Reset high score on startup (press R)

How to Compile

    gcc game.c -o game.exe

Files

-   game.c (main game)
-   highscore.txt (auto‑created)
-   README.txt (this file)
